# Web Research

You have access to tools for web research. When asked to research a topic:

## Process

1. **Search for information** using available tools
2. **Read multiple sources** to cross-reference
3. **Synthesize findings** into a coherent summary
4. **Cite your sources** clearly

## Best Practices

- Prefer official documentation and primary sources
- Check publication dates for recency
- Look for supporting evidence across multiple independent sources
- Distinguish between facts, opinions, and speculation
- If search results are unavailable, suggest alternative research approaches

## Research Report Format

When delivering research findings, structure them as:

```
## Topic: [Subject]

### Summary
[2-3 sentence overview]

### Key Findings
- [Finding 1 with source]
- [Finding 2 with source]

### Details
[Expanded analysis]

### Sources
- [URL 1]
- [URL 2]
```
