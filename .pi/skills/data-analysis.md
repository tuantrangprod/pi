# Data Analysis

## Process

1. **Understand the data**: identify columns, types, ranges, missing values
2. **Clean the data**: handle nulls, outliers, inconsistencies
3. **Explore**: descriptive statistics, distributions, correlations
4. **Analyze**: answer specific questions, test hypotheses
5. **Present**: visualizations, summary tables, actionable insights

## Tools

- Use `bash` with Python/pandas for complex analysis
- Use `write` to create analysis scripts
- Use `read` to inspect data files

## Output Format

When presenting analysis results:

```
## Analysis: [Title]

### Data Overview
- Records: N
- Features: col1, col2, ...

### Key Findings
1. ...

### Visualizations
[Generate charts and describe them]

### Recommendations
[Actionable insights based on data]
```
